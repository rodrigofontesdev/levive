# Clínica Levive

> [!NOTE]
> This site is under development!

# Pré-requisitos

- Git
- Node >= v20

# Como iniciar

Clone o repositório em um novo diretório:

```sh
git clone git@github.com:rodrigofontesdev/levive.git
```

```sh
cd levive
```

Instale as dependências do projeto:

```sh
npm install
```

Inicie o Next.js:

```sh
npm run dev
```

# Construído com

- Next.js v15
- TypeScript
- Tailwind CSS
